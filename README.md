# Kidney Disease Classification using Deep Learning, Flask & MLflow

An end-to-end deep learning project for Kidney Disease Classification with model training, prediction pipeline, experiment tracking, and web deployment.

## Features
- Deep learning image classification model
- Flask-based prediction web application
- MLflow experiment tracking and logging
- Modular end-to-end ML pipeline
- Model evaluation and inference pipeline

## Tech Stack
- Python
- TensorFlow / Keras
- Flask
- MLflow
- NumPy
- Pandas

## Future Enhancements
- Docker integration
- AWS deployment
- CI/CD pipeline
